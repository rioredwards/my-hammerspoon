# Hammerspoon Configuration

Personal Hammerspoon setup. Architecture, conventions, and how to add features: [AGENTS.md](AGENTS.md).

## Status Menu

- The status menubar menu shows each feature as a checkbox. Click a feature to toggle it on or off, and Hammerspoon reloads with that choice persisted in `hs.settings`.
- Registered `hammerspoon://...` deeplinks appear under the `Deeplinks` submenu.
